# Cafe Site Project
お洒落なカフェの雰囲気を伝えるためのコンセプトサイトです。
レスポンシブデザインに対応し、どのデバイスからでも見やすいレイアウトを実現しました。

## 🚀 公開URL
https://colorful01sky-creator.github.io/cafe-site/

## 🛠 使用技術
- HTML5 / CSS3 (レスポンシブ対応)
- レスポンシブデザイン（PC・タブレット・スマホ対応）
- GitHub Pages（ホスティング）

## ✨ こだわったポイント
- **世界観の統一**: カフェのコンセプトに合わせ、配色やフォント、画像の選定にこだわり、落ち着いた雰囲気を演出しました。
- **ユーザビリティ**: 直感的にメニューやアクセス情報に辿り着けるよう、シンプルで分かりやすいナビゲーションを設計しました。
- **丁寧なコーディング**: 誰が見ても理解しやすいよう、セクションごとにコメントを入れ、整理されたコードを意識しました。

## 📂 フォルダ構成
- `index.html`: メインページ
- `css`: スタイルシート（レイアウト用、装飾用）
- `image/`: サイト内で使用している画像一式
